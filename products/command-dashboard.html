Overview

What it does

The Live Streaming & Command Dashboard is where every AMS Safe City module comes together. It aggregates all camera feeds into a single, searchable, real-time operator interface — complete with alert history, detection logs, map-based incident tracking, and response-coordination tools.

This is the interface layer of the platform's three-layer architecture: every detection from every module — weapons, faces, crowds, breaches and more — surfaces here for review by operators, police dispatch, and city administrators, all under strict role-based access control.
